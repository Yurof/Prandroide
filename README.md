# Prandroide
Apprentissage et généralisation sur une tâche de navigation d'un robot à roues 

## installation
lancer `install-dependencies.sh` pour installer les librairies nécessaires :
- gym
- libfastsim
- pybind11
- pyfastsim
- pybullet-gym

## Usage

`python Simulationfastsim/RobotAléatoire.py`
